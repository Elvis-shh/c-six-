### 2.3 实施步骤

**步骤 1：后端洞察生成**

```java
public PredictInsightResponse getInsights(String companyCode) {
    PredictResponse predict = predictService.predict(companyCode);
    Company company = companyRepo.findByCode(companyCode);

    // 计算 CAGR
    BigDecimal revenueCagr = indicatorService.calculateCAGR(getRevenueHistory(companyCode), 5);
    BigDecimal profitCagr = indicatorService.calculateCAGR(getProfitHistory(companyCode), 5);

    return PredictInsightResponse.builder()
        .revenueTrend(String.format(
            "基于近5年复合增长率约%.0f%%，预计2025年营收将达到%.0f亿，2026年有望突破%.0f亿。",
            revenueCagr, getPredictValue(predict, "revenue", 2025), getPredictValue(predict, "revenue", 2026)))
        .profitTrend(String.format(
            "净利润预计保持%.0f%%左右增速，2026年归母净利润有望达到%.0f亿。",
            profitCagr, getPredictValue(predict, "profit", 2026)))
        .assumptions(List.of(
            "行业政策环境保持稳定",
            "公司市场份额不发生重大变化",
            "宏观经济不出现严重衰退"
        ))
        .riskNote("以上预测基于历史数据趋势外推，未考虑突发事件影响。实际业绩可能因市场、政策、竞争等因素而与预测值存在较大偏差。")
        .disclaimer("本预测仅为基于历史数据的统计分析，不构成任何投资建议。预测值不代表未来实际业绩，投资者应独立判断。")
        .build();
}
```
