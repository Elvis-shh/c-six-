## 一、PRD（产品需求文档）

### 1.1 背景

财报数据本身是冰冷的数字，用户需要专家视角的解读。系统应自动识别经营亮点，让用户快速抓住公司的核心竞争力。

### 1.2 需求描述

作为用户，我希望能看到系统自动识别出的公司经营亮点，以绿色卡片形式展示。

### 1.3 验收标准

| # | 验收条件 |
|---|---------|
| AC1 | Dashboard 中渲染经营亮点区域，含 2-4 张亮点卡片 |
| AC2 | 每张卡片绿色左边框 + emoji icon + 标题 + 描述 |
| AC3 | 描述文案自动填充公司实际数值（如"毛利率超 92.8%"） |
| AC4 | 不同公司数据触发不同的亮点（茅台→品牌护城河，银行→资产规模） |
| AC5 | 无匹配规则时显示"暂无亮点数据" |

**步骤 2：实现 ConditionEvaluator**

```java
public class ConditionEvaluator {
    public boolean evaluate(String expression, Map<String, BigDecimal> context) {
        // 简单条件解析：支持 > < >= <= == AND OR
        // "grossMargin > 80 AND netMargin > 40"
        // 拆分为子条件，逐个求值
        String[] andParts = expression.split("AND");
        for (String part : andParts) {
            if (!evaluateSimple(part.trim(), context)) return false;
        }
        return true;
    }

    private boolean evaluateSimple(String expr, Map<String, BigDecimal> context) {
        Pattern p = Pattern.compile("(\\w+)\\s*(>|<|>=|<=|==)\\s*([\\d.]+)");
        Matcher m = p.matcher(expr);
        if (m.find()) {
            String var = m.group(1);
            String op = m.group(2);
            BigDecimal threshold = new BigDecimal(m.group(3));
            BigDecimal actual = context.getOrDefault(var, BigDecimal.ZERO);
            return switch (op) {
                case ">" -> actual.compareTo(threshold) > 0;
                case "<" -> actual.compareTo(threshold) < 0;
                case ">=" -> actual.compareTo(threshold) >= 0;
                case "<=" -> actual.compareTo(threshold) <= 0;
                case "==" -> actual.compareTo(threshold) == 0;
                default -> false;
            };
        }
        return false;
    }
}
```

**步骤 3：实现 HighlightService**

```java
public List<HighlightItem> getHighlights(String companyCode) {
    // 1. 获取公司数据和行业均值
    Map<String, BigDecimal> context = buildContext(companyCode);

    // 2. 查询所有启用的规则
    List<HighlightRule> rules = ruleRepo.findByEnabledTrueOrderByPriority();

    // 3. 逐条评估
    return rules.stream()
        .filter(r -> conditionEvaluator.evaluate(r.getConditionExpr(), context))
        .map(r -> HighlightItem.builder()
            .icon(r.getIcon())
            .title(r.getTitle())
            .description(templateEngine.render(r.getDescTemplate(), context))
            .ruleKey(r.getRuleKey())
            .build())
        .toList();
}
```
