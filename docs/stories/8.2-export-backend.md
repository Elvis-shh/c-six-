# Story 8.2 — 导出水印与合规（后端实现）

> **Epic**: export (报告导出) | **优先级**: P2 | **阶段**: Phase 2 核心闭环
> **依赖**: Story 8.1 | **预估工时**: 1.5d

---

## 一、PRD（产品需求文档）

### 1.1 背景

前端导出方案适合快速使用，但保真度和合规性不足。服务端导出可生成高质量 PDF（含水印）、Word/Excel（含模板化免责声明）。

### 1.2 需求描述

作为用户，我可以通过服务端导出获得更高质量的报告文件，且自动包含水印和合规声明。

### 1.3 验收标准

| # | 验收条件 |
|---|---------|
| AC1 | `POST /api/v1/export` 提交导出任务，返回 taskId |
| AC2 | `GET /api/v1/export/tasks/{taskId}` 轮询任务状态 |
| AC3 | `GET /api/v1/export/download/{taskId}` 下载完成文件 |
| AC4 | PDF 包含"仅供参考"半透明水印 |
| AC5 | Word/Excel 页脚包含免责声明 |
| AC6 | 导出文件存储到 MinIO，24 小时后自动清理 |

### 1.4 API 规格

**POST `/api/v1/export`**
```json
// Request
{ "companyCode": "600519", "format": "pdf", "includeModules": ["overview","predict","risks"] }

// Response 202
{ "code": 0, "data": { "taskId": "export_def456", "status": "pending" } }
```

**GET `/api/v1/export/tasks/export_def456`**
```json
// Response (进行中)
{ "code": 0, "data": { "taskId": "export_def456", "status": "rendering", "progress": 60 } }

// Response (完成)
{ "code": 0, "data": { "taskId": "export_def456", "status": "ready", "downloadUrl": "/api/v1/export/download/export_def456" } }
```

## 二、开发方案

### 2.1 技术方案

| 技术点 | 选型 | 说明 |
|--------|------|------|
| PDF 生成 | Flying Saucer / iText | Java 端 HTML→PDF |
| Word 生成 | Apache POI / docx4j | 模板 + 动态数据填充 |
| Excel 生成 | Apache POI | 表格数据导出 |
| 文件存储 | MinIO | 临时存储，24h TTL |
| 异步任务 | Spring `@Async` / RocketMQ | MVP 用 `@Async`，P3 升级为 MQ |
| 水印 | CSS `position:fixed` + opacity | PDF 渲染时叠加 |

### 2.3 实施步骤

**步骤 1：ExportController**

```java
@RestController
@RequestMapping("/api/v1/export")
public class ExportController {

    @PostMapping
    public ApiResponse<ExportTaskResponse> submitExport(@RequestBody ExportRequest req) {
        String taskId = exportService.submit(req);
        return ApiResponse.accepted(new ExportTaskResponse(taskId, "pending"));
    }

    @GetMapping("/tasks/{taskId}")
    public ApiResponse<ExportTaskResponse> getTaskStatus(@PathVariable String taskId) {
        return ApiResponse.success(exportService.getTaskStatus(taskId));
    }

    @GetMapping("/download/{taskId}")
    public ResponseEntity<Resource> download(@PathVariable String taskId) {
        ExportRecord record = exportService.getCompletedTask(taskId);
        InputStreamResource resource = minioService.download(record.getFileUrl());
        return ResponseEntity.ok()
            .header(HttpHeaders.CONTENT_DISPOSITION, "attachment; filename=" + record.getFileName())
            .contentType(MediaType.APPLICATION_OCTET_STREAM)
            .body(resource);
    }
}
```

**步骤 2：ExportService**

```java
@Service
public class ExportService {

    @Async
    public void processExport(ExportRecord record) {
        try {
            record.setStatus("rendering");
            exportRecordRepo.save(record);

            byte[] fileBytes = switch (record.getFormat()) {
                case "pdf" -> generatePdf(record);
                case "word" -> generateWord(record);
                case "excel" -> generateExcel(record);
                default -> throw new IllegalArgumentException("Unsupported format");
            };

            // 上传到 MinIO
            String fileUrl = minioService.upload(fileBytes, buildObjectKey(record));
            record.setFileUrl(fileUrl);
            record.setStatus("ready");
        } catch (Exception e) {
            log.error("Export failed: {}", record.getTaskId(), e);
            record.setStatus("failed");
        }
        exportRecordRepo.save(record);
    }

    byte[] generatePdf(ExportRecord record) {
        String html = buildReportHtml(record);  // 渲染完整 HTML
        // 添加水印 CSS
        html = html.replace("</body>",
            "<div style='position:fixed;top:0;left:0;width:100%;height:100%;" +
            "display:flex;align-items:center;justify-content:center;" +
            "pointer-events:none;opacity:0.06;font-size:48px;transform:rotate(-30deg);" +
            "color:#000;'>仅供参考</div></body>");
        // 使用 Flying Saucer 渲染
        return pdfRenderer.render(html);
    }

    byte[] generateWord(ExportRecord record) {
        // 使用 Apache POI XWPF 生成
        XWPFDocument doc = new XWPFDocument();
        // ... 添加段落、表格
        // 添加页脚免责声明
        XWPFFooter footer = doc.createFooter(HeaderFooterType.DEFAULT);
        XWPFParagraph p = footer.createParagraph();
        p.setAlignment(ParagraphAlignment.CENTER);
        XWPFRun run = p.createRun();
        run.setText("免责声明：本报告由 SmartReport 自动生成，仅供学习参考，不构成任何投资建议。");
        run.setFontSize(9);
        // 写入 ByteArrayOutputStream 返回
        return toByteArray(doc);
    }
}
```

**步骤 3：定期清理过期文件**

```java
@Scheduled(cron = "0 0 2 * * ?")  // 每天凌晨 2 点
public void cleanExpiredExports() {
    List<ExportRecord> expired = exportRecordRepo
        .findByStatusAndCreatedAtBefore("ready", LocalDateTime.now().minusHours(24));
    for (ExportRecord record : expired) {
        minioService.delete(record.getFileUrl());
        record.setStatus("expired");
        exportRecordRepo.save(record);
    }
}
```
