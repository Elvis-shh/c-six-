# Story 3.3 — 折线图趋势展示

> **Epic**: dashboard (财报总览看板) | **优先级**: P1 | **阶段**: Phase 1 MVP
> **依赖**: Story 3.2 | **预估工时**: 1.5d

---

## 一、PRD（产品需求文档）

### 1.1 背景

表格数据是精确的，但趋势不够直观。用户需要在折线图模式下查看单个指标的 5 年趋势，并自由切换指标。

### 1.2 需求描述

作为用户，我可以切换到折线图模式，查看单个指标的 5 年趋势，并自由切换指标。

### 1.3 验收标准

| # | 验收条件 |
|---|---------|
| AC1 | 从表格切换到折线图 → Chart.js 折线图渲染 |
| AC2 | 折线图显示 X 轴（年份）+ Y 轴（指标值 + 单位） |
| AC3 | 指标切换 Tab 按钮组（营收/净利润/毛利率/资产负债率/现金流） |
| AC4 | 点击不同 Tab → 折线图切换为对应指标数据 |
| AC5 | Tooltip 悬停显示：年份、指标名、数值、单位 |
| AC6 | 切换公司 → 旧 chart destroy + 新 chart 创建，无内存泄漏 |

## 二、开发方案

### 2.1 技术方案

| 技术点 | 选型 | 说明 |
|--------|------|------|
| 图表库 | Chart.js 4 + vue-chartjs | 轻量、Canvas 渲染、支持响应式 |
| 实例管理 | `useChart` composable | 封装 create/update/destroy 生命周期 |
| 动态数据 | `watch` 监听 metricKey 变化 | 更新 chart.data.datasets |
| 内存泄漏防护 | `onBeforeUnmount` + `chart.destroy()` | 组件卸载时销毁实例 |

### 2.2 关键文件

| 文件 | 说明 |
|------|------|
| `frontend/src/components/TrendChart.vue` | 折线图组件 |
| `frontend/src/components/MetricTabs.vue` | 指标切换按钮组 |
| `frontend/src/composables/useChart.ts` | Chart.js 实例管理 |

### 2.3 实施步骤

**步骤 1：安装依赖**

```bash
npm install chart.js vue-chartjs
```

**步骤 2：实现 useChart composable**

```typescript
// composables/useChart.ts
import { Chart, LineController, LineElement, PointElement, LinearScale,
         CategoryScale, Tooltip, Filler, Legend } from 'chart.js'

Chart.register(LineController, LineElement, PointElement, LinearScale,
               CategoryScale, Tooltip, Filler, Legend)

export function useChart(canvasRef: Ref<HTMLCanvasElement | null>) {
  let chartInstance: Chart | null = null

  function create(config: any) {
    destroy()
    if (canvasRef.value) {
      chartInstance = new Chart(canvasRef.value, config)
    }
  }

  function updateData(labels: string[], datasets: any[]) {
    if (chartInstance) {
      chartInstance.data.labels = labels
      chartInstance.data.datasets = datasets
      chartInstance.update('default')
    }
  }

  function destroy() {
    if (chartInstance) {
      chartInstance.destroy()
      chartInstance = null
    }
  }

  onBeforeUnmount(destroy)

  return { create, updateData, destroy }
}
```

**步骤 3：实现 TrendChart.vue**

```vue
<script setup lang="ts">
const props = defineProps<{
  years: string[]
  metric: MetricSeries | null
}>()

const canvasRef = ref<HTMLCanvasElement | null>(null)
const { create, updateData } = useChart(canvasRef)

watch(() => props.metric, (m) => {
  if (!m) return
  updateData(props.years, [{
    label: m.name,
    data: m.values,
    borderColor: '#4F8CF7',
    backgroundColor: 'rgba(79,140,247,0.1)',
    fill: true,
    tension: 0.3,
    pointRadius: 5,
    pointHoverRadius: 7,
  }])
}, { immediate: true })

onMounted(() => {
  create({
    type: 'line',
    data: { labels: [], datasets: [] },
    options: {
      responsive: true,
      maintainAspectRatio: false,
      plugins: {
        tooltip: {
          callbacks: {
            label: (ctx) => `${ctx.dataset.label}: ${ctx.parsed.y.toLocaleString()} ${props.metric?.unit || ''}`
          }
        },
        legend: { display: false },
      },
      scales: {
        y: {
          beginAtZero: false,
          ticks: { callback: (v) => v.toLocaleString() }
        }
      }
    }
  })
})
</script>

<template>
  <div class="chart-container">
    <canvas ref="canvasRef" />
  </div>
</template>

<style scoped>
.chart-container {
  position: relative;
  height: 350px;
  width: 100%;
}
</style>
```

**步骤 4：实现 MetricTabs.vue**

```vue
<script setup lang="ts">
defineProps<{
  metrics: { key: string, name: string }[]
  modelValue: string
}>()
defineEmits<{ 'update:modelValue': [key: string] }>()
</script>

<template>
  <div class="metric-tabs">
    <button
      v-for="m in metrics" :key="m.key"
      :class="{ active: modelValue === m.key }"
      @click="$emit('update:modelValue', m.key)"
    >{{ m.name }}</button>
  </div>
</template>
```
