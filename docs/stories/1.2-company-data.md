## 一、PRD（产品需求文档）

### 1.1 背景

搜索是用户进入 SmartReport 的第一入口。需要预置 A 股上市公司基础信息，并提供模糊搜索接口，让用户通过公司名或股票代码快速找到目标公司。

### 1.2 需求描述

作为用户，我可以在搜索框输入公司名或股票代码，获得实时建议并快速加载分析。

### 1.3 验收标准

| # | 验收条件 | 验证方式 |
|---|---------|---------|
| AC1 | `GET /api/v1/search/companies?q=茅台&limit=8` 返回贵州茅台、茅台相关公司 | API 测试 |
| AC2 | `GET /api/v1/search/companies?q=600519` 返回贵州茅台（代码匹配） | API 测试 |
| AC3 | `GET /api/v1/search/companies/hot` 返回 6 家热门公司 | API 测试 |
| AC4 | 搜索无匹配时返回空数组（非 404） | API 测试 |
| AC5 | 搜索不分大小写、支持拼音首字母（可选） | API 测试 |
| AC6 | 前端 TypeScript 类型与后端响应一致 | 编译检查 |

### 2.3 实施步骤

**步骤 1：扩展 CompanyRepository**

```java
@Repository
public interface CompanyRepository extends JpaRepository<Company, String> {
    // 模糊匹配公司名或股票代码
    @Query("SELECT c FROM Company c WHERE c.status = 1 AND " +
           "(c.name LIKE CONCAT('%', :keyword, '%') OR " +
           "c.code LIKE CONCAT('%', :keyword, '%') OR " +
           "c.shortName LIKE CONCAT('%', :keyword, '%'))")
    Page<Company> searchCompanies(@Param("keyword") String keyword, Pageable pageable);

    // 热门公司（按搜索频率排序，初期硬编码）
    List<Company> findByCodeIn(List<String> codes);
}
```

**步骤 2：实现 SearchService**

- `searchCompanies(keyword, limit)` → 调用 Repository 分页查询
- `getHotCompanies()` → 返回 6 家预设热门公司代码
- 异常处理：空参数、超长字符串

**步骤 3：实现 SearchController**

```java

@RequestMapping("/api/v1/search")
public class SearchController {
    @GetMapping("/companies")
    public ApiResponse<List<SearchCompanyResponse>> search(
        @RequestParam String q,
        @RequestParam(defaultValue = "8") int limit) { ... }

    @GetMapping("/companies/hot")
    public ApiResponse<List<SearchCompanyResponse>> hot() { ... }
}
```

**步骤 4：编写数据导入脚本**

`ai-engine/app/services/data_importer.py`:
- 使用 AKShare/Tushare 获取 A 股上市公司列表
- 数据清洗：统一行业分类、去除退市公司
- 生成 INSERT SQL 或直接通过 JDBC 写入 MySQL
- 支持增量更新

**步骤 5：前端类型定义 + API 层**

`frontend/src/types/index.ts`:
```typescript
export interface Company {
  code: string
  name: string
  shortName: string
  industry: string
  market: 'SH' | 'SZ' | 'BJ'
}

export interface SearchResponse {
  code: number
  data: Company[]
}
```

`frontend/src/api/index.ts`:
```typescript
import axios from 'axios'

const api = axios.create({ baseURL: '/api/v1' })

export const searchCompanies = (q: string, limit = 8) =>
  api.get<SearchResponse>('/search/companies', { params: { q, limit } })

export const getHotCompanies = () =>
  api.get<SearchResponse>('/search/companies/hot')
```
