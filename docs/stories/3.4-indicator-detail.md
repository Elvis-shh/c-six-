## 一、PRD（产品需求文档）

### 1.1 背景

普通用户可能不理解"毛利率"、"ROE"等专业术语的含义。指标详解表提供每个指标的数值、变化趋势、行业对比和评价，并用白话解释帮助用户理解。

### 1.2 需求描述

作为用户，我可以查看带术语解释的指标详情表，了解每个指标的含义和行业对比。

### 1.3 验收标准

| # | 验收条件 |
|---|---------|
| AC1 | 渲染 5 个核心指标详解行（毛利率、净利率、资产负债率、经营现金流、ROE） |
| AC2 | 每行包含：指标名（带 ? Tooltip）、当前值、同比变化、行业均值、行业对比、评价标签 |
| AC3 | 鼠标悬浮 ? 图标 → 显示白话术语解释气泡 |
| AC4 | 评价标签颜色：优秀=绿、良好=蓝、健康=绿、关注=橙、警示=红 |
| AC5 | 同比变化箭头颜色遵循金融惯例（同 Story 3.1） |

## 二、开发方案

### 2.1 技术方案

| 技术点 | 选型 | 说明 |
|--------|------|------|
| Tooltip | 自研 `<TermTooltip>` 组件 | 基于 `position: absolute` + hover 事件 |
| 术语数据 | `GET /api/v1/terms` | 返回全量术语字典 |
| 评价标签 | 枚举映射 `ratingMap` | `{ excellent: '优秀', good: '良好', healthy: '健康', warning: '关注', danger: '警示' }` |
| 颜色映射 | CSS 变量 | `--rating-excellent: #22c55e` 等 |

### 2.2 关键文件

| 文件 | 说明 |
|------|------|
| `frontend/src/components/IndicatorDetail.vue` | 指标详解表主组件 |
| `frontend/src/components/TermTooltip.vue` | 术语悬浮提示 |
| `frontend/src/components/RatingTag.vue` | 评价标签组件 |
| `frontend/src/api/index.ts` | getReportIndicators, getTerms API |

### 2.3 实施步骤

**步骤 1：后端接口**

```java
// GET /api/v1/reports/{companyCode}/indicators
// 返回 5 个核心指标 + 行业对比 + 评价

@GetMapping("/{companyCode}/indicators")
public ApiResponse<List<IndicatorDetailResponse>> getIndicators(@PathVariable String companyCode) {
    return ApiResponse.success(indicatorService.getDetailedIndicators(companyCode));
}
```

**步骤 3：实现 RatingTag.vue**

```vue
<script setup lang="ts">
import { computed } from 'vue'

const props = defineProps<{ rating: string }>()

const colorMap: Record<string, string> = {
  excellent: '#22c55e',
  good: '#3b82f6',
  healthy: '#22c55e',
  warning: '#f59e0b',
  danger: '#ef4444',
}
const labelMap: Record<string, string> = {
  excellent: '优秀', good: '良好', healthy: '健康',
  warning: '关注', danger: '警示',
}

const bgColor = computed(() => colorMap[props.rating] || '#6b7280')
const label = computed(() => labelMap[props.rating] || props.rating)
</script>

<template>
  <span class="rating-tag" :style="{ background: bgColor + '20', color: bgColor, borderColor: bgColor }">
    {{ label }}
  </span>
</template>
```
