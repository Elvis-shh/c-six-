## 一、PRD（产品需求文档）

### 1.1 背景

用户不仅关心过去的财报数据，也想知道公司未来的发展趋势。基于历史数据的趋势外推可以帮助用户形成预期。

### 1.2 需求描述

作为用户，我可以查看基于历史数据外推的营收和净利润预测图表（未来 2 年）。

### 1.3 验收标准

| # | 验收条件 |
|---|---------|
| AC1 | 预测模块独立 Tab（"趋势预测"），与"财报总览"并列 |
| AC2 | 双折线图：实际值（2020-2024 实线）+ 预测值（2025E-2026E 虚线）|
| AC3 | 预测区间半透明色带表示置信区间 |

### 2.3 实施步骤

**步骤 1：实现预测算法**

```java
public class PredictServiceImpl implements PredictService {

    public PredictResponse predict(String companyCode) {
        // 获取近 5 年营收和净利润数据
        List<BigDecimal> revenues = getHistoricalValues(companyCode, "revenue", 5);
        List<BigDecimal> profits = getHistoricalValues(companyCode, "profit", 5);

        // 线性回归预测
        LinearRegressionResult revResult = linearRegression(revenues);
        LinearRegressionResult profResult = linearRegression(profits);

        // 构建响应
        return PredictResponse.builder()
            .years(List.of("2020","2021","2022","2023","2024","2025E","2026E"))
            .series(List.of(
                PredictSeries.builder().key("revenue_actual").name("营收（实际）")
                    .type("solid").values(toList(revenues, null, null)).build(),
                PredictSeries.builder().key("revenue_predict").name("营收（预测）")
                    .type("dashed").values(toListWithPredict(revenues, revResult)).build(),
                // 净利润同理...
            ))
            .build();
    }

    private LinearRegressionResult linearRegression(List<BigDecimal> values) {
        int n = values.size();
        double sumX = 0, sumY = 0, sumXY = 0, sumX2 = 0;
        for (int i = 0; i < n; i++) {
            double x = i;
            double y = values.get(i).doubleValue();
            sumX += x; sumY += y;
            sumXY += x * y; sumX2 += x * x;
        }
        double slope = (n * sumXY - sumX * sumY) / (n * sumX2 - sumX * sumX);
        double intercept = (sumY - slope * sumX) / n;
        // 预测 n+1, n+2 期
        BigDecimal predict1 = BigDecimal.valueOf(intercept + slope * n);
        BigDecimal predict2 = BigDecimal.valueOf(intercept + slope * (n + 1));
        // 计算标准误差（置信区间）
        double stdError = computeStdError(values, slope, intercept);
        return new LinearRegressionResult(predict1, predict2, stdError);
    }
}
```

**步骤 3：前端 PredictChart.vue**

```vue
<script setup lang="ts">
const props = defineProps<{ data: PredictResponse }>()

const chartData = computed(() => ({
  labels: props.data.years,
  datasets: props.data.series.map(s => ({
    label: s.name,
    data: s.values,
    borderDash: s.type === 'dashed' ? [6, 4] : [],
    borderColor: s.key.includes('revenue') ? '#4F8CF7' : '#22C55E',
    backgroundColor: s.type === 'dashed' ? 'rgba(79,140,247,0.08)' : 'transparent',
    fill: s.type === 'dashed',
    tension: 0.3,
    pointRadius: s.type === 'dashed' ? 4 : 5,
    pointStyle: s.type === 'dashed' ? 'rectRounded' : 'circle',
    borderWidth: 2,
  }))
}))

const chartOptions = {
  responsive: true,
  maintainAspectRatio: false,
  plugins: {
    tooltip: {
      callbacks: {
        label: (ctx) => {
          const val = ctx.parsed.y
          return val ? `${ctx.dataset.label}: ${val.toLocaleString()} 亿` : ''
        }
      }
    },
    legend: { position: 'bottom' as const }
  },
  scales: {
    y: {
      beginAtZero: false,
      ticks: { callback: (v: number) => v.toLocaleString() + ' 亿' }
    }
  }
}
</script>
```
