# Story 2.1 — 统一搜索入口

> **Epic**: search-load (公司搜索与加载) | **优先级**: P0 | **阶段**: Phase 1 MVP
> **依赖**: Story 1.2 | **预估工时**: 2d

---

## 一、PRD（产品需求文档）

### 1.1 背景

搜索入口是用户接触 SmartReport 的第一个交互界面。需要实现一个体验流畅的搜索框：实时建议、键盘导航、空状态处理。

### 1.2 需求描述

作为用户，我可以在搜索框输入公司名或股票代码，获得实时建议并快速加载分析。

### 1.3 验收标准

| # | 验收条件 |
|---|---------|
| AC1 | 输入字符后 300ms 出现搜索建议下拉列表 |
| AC2 | 下拉列表每项显示：公司名、股票代码、行业标签 |
| AC3 | 搜索关键词在结果中高亮显示（黄色背景或加粗） |
| AC4 | ↑↓ 键切换选中项，Enter 确认跳转，Esc 关闭下拉 |
| AC5 | 无匹配结果时显示"未找到匹配公司"空状态 |
| AC6 | 搜索框为空时不显示下拉，首次聚焦可显示热门推荐 |
| AC7 | 快速连续输入时，仅最后一次请求生效（取消前一次） |

### 1.4 交互说明

```
┌─────────────────────────────────────┐
│  🔍 搜索公司名称或股票代码...       │  ← SearchBox
│                                     │
│  ┌─────────────────────────────────┐│
│  │ 🏢 贵州茅台   600519  白酒      ││  ← 键盘 ↑↓ 切换高亮
│  │ 🏢 五粮液     000858  白酒      ││     背景色区分
│  │ 🏢 宁德时代   300750  新能源    ││
│  └─────────────────────────────────┘│
│                                     │
│  未找到匹配公司                     │  ← 空状态（搜索无结果时）
└─────────────────────────────────────┘
```

**键盘导航**：
- `↑` / `↓`：在建议列表中上下移动高亮项（循环）
- `Enter`：选择当前高亮项，跳转 Dashboard
- `Esc`：关闭建议下拉，清空搜索框
- 鼠标悬停：对应项高亮，点击选择

### 2.1 技术方案

|--------|------|------|
| 搜索组件 | Vue 3 Composition API | `<script setup>` + TypeScript |

### 2.3 实施步骤

**步骤 1：创建 searchStore**

```typescript
// stores/searchStore.ts
export const useSearchStore = defineStore('search', () => {
  const query = ref('')
  const results = ref<Company[]>([])
  const isLoading = ref(false)
  const selectedIndex = ref(-1)
  const isOpen = ref(false)

  async function search(q: string) {
    query.value = q
    if (!q.trim()) { results.value = []; isOpen.value = false; return }
    isLoading.value = true
    try {
      const res = await searchCompanies(q, 8)
      results.value = res.data.data
      isOpen.value = true
      selectedIndex.value = -1
    } finally {
      isLoading.value = false
    }
  }

  return { query, results, isLoading, selectedIndex, isOpen, search }
})
```

**步骤 2：创建 useSearch composable**

```typescript
// composables/useSearch.ts
export function useSearch() {
  const store = useSearchStore()
  const router = useRouter()
  let abortController: AbortController | null = null

  const debouncedSearch = useDebounceFn((q: string) => {
    if (abortController) abortController.abort()
    abortController = new AbortController()
    store.search(q)
  }, 300)

  function handleKeydown(e: KeyboardEvent) {
    if (!store.isOpen) return
    if (e.key === 'ArrowDown') {
      store.selectedIndex = Math.min(store.selectedIndex + 1, store.results.length - 1)
    } else if (e.key === 'ArrowUp') {
      store.selectedIndex = Math.max(store.selectedIndex - 1, 0)
    } else if (e.key === 'Enter' && store.selectedIndex >= 0) {
      selectCompany(store.results[store.selectedIndex])
    } else if (e.key === 'Escape') {
      store.isOpen = false
    }
  }

  function selectCompany(company: Company) {
    store.isOpen = false
    router.push(`/dashboard/${company.code}`)
  }

  return { debouncedSearch, handleKeydown, selectCompany }
}
```

### 2.5 高亮匹配实现

```typescript
export function highlightMatch(text: string, keyword: string): string {
  if (!keyword.trim()) return text
  const escaped = keyword.replace(/[.*+?^${}()|[\]\\]/g, '\\$&')
  const regex = new RegExp(`(${escaped})`, 'gi')
  return text.replace(regex, '<mark class="search-highlight">$1</mark>')
}
```
