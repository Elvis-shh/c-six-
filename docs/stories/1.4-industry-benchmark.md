## 一、PRD（产品需求文档）

### 1.1 背景

用户在看财报指标时，需要知道公司在同行业中的位置。"毛利率 60%"本身意义有限，但"毛利率行业前 10%"让用户一目了然。

### 1.2 需求描述

作为用户，我可以在指标详解表中看到每个指标的行业均值对比和行业排名，了解公司在同行业中的位置。

### 1.3 验收标准

| # | 验收条件 | 验证方式 |
|---|---------|---------|
| AC1 | `GET /api/v1/analysis/600519/benchmark` 返回白酒行业各指标均值 | API 测试 |
| AC2 | 响应中包含行业排名描述："毛利率行业前 10%" | API 测试 |
| AC3 | 前端指标详解表渲染"行业均值"列和"行业对比"列 | 页面检查 |
| AC4 | 切换不同行业公司 → 行业均值数据相应变化 | 页面检查 |

### 1.5 UI 说明

在 Dashboard 的"核心财务指标详解表"(IndicatorDetail.vue) 中：
- 新增"行业均值"列：显示当前行业该指标的平均值
- 新增"行业对比"列：显示排名标签（如"行业前 10%"），用不同颜色区分优劣

## 二、开发方案

### 2.1 技术方案

| 技术点 | 选型 | 说明 |
|--------|------|------|
| 行业分类 | 证监会行业分类 | 从 `companies.industry` 字段获取 |
| 均值数据 | `industry_averages` 表 | 预计算存储，避免实时聚合 |
| 排名算法 | 分位数计算 | `(1 - companyValue / industryPercentile) * 100` |
| 缓存策略 | Redis | 行业均值变更频率极低，缓存 24h |

### 2.2 关键文件

| 文件 | 说明 |

| `backend/.../repository/IndustryAverageRepository.java` | 行业均值查询 |
| `frontend/src/components/IndicatorDetail.vue` | 指标详解表（新增两列） |

**步骤 2：实现 BenchmarkService**

```java
public class BenchmarkServiceImpl implements BenchmarkService {

    public BenchmarkResponse getBenchmark(String companyCode, int year) {
        Company company = companyRepo.findByCode(companyCode);
        List<IndustryAverage> averages = industryAverageRepo
            .findByIndustryAndYear(company.getIndustry(), year);

        return BenchmarkResponse.builder()
            .industry(company.getIndustry())
            .indicators(averages.stream().map(avg -> {
                FinancialIndicator companyIndicator = indicatorRepo
                    .findByCompanyCodeAndKeyAndYear(companyCode, avg.getIndicatorKey(), year);
                return buildIndicatorBenchmark(avg, companyIndicator);
            }).toList())
            .build();
    }

    private String computeRank(BigDecimal companyValue, BigDecimal avgValue, BigDecimal medianValue) {
        // 简单分档：前5%/前10%/前25%/前50%/后50%
        if (companyValue.compareTo(avgValue.multiply(new BigDecimal("1.5"))) > 0) return "top5%";
        if (companyValue.compareTo(avgValue.multiply(new BigDecimal("1.2"))) > 0) return "top10%";
        // ...
    }
}
```
