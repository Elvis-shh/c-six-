## 一、PRD（产品需求文档）

### 1.1 背景

用户在查看财报数据时，会产生各种问题（"这家公司盈利能力怎么样？"、"有什么风险？"）。智能问答面板提供 AI 对话能力，让用户可以直接提问并获得即时回答。

### 1.2 需求描述

作为用户，我可以在浮动面板中与 AI 助手进行财报相关问答，包括发送消息、查看历史回复和建议问题。

### 1.3 验收标准

| # | 验收条件 |
|---|---------|
| AC1 | 右下角浮动按钮 💬，点击展开聊天面板（从右侧滑入） |
| AC2 | 面板含：消息列表区、输入框、发送按钮 |
| AC3 | 用户消息显示在右侧蓝色气泡，助手回复显示在左侧灰色气泡 |
| AC4 | 首次打开显示欢迎语 + 3 个建议问题快捷按钮 |
| AC5 | 发送消息后自动滚动到最新消息 |
| AC6 | 支持 Enter 发送，Shift+Enter 换行 |
| AC7 | 浮动按钮支持拖拽移动 |

## 二、开发方案

### 2.1 技术方案

| 技术点 | 选型 | 说明 |
|--------|------|------|
| 面板动画 | CSS `transform: translateX()` + `transition` | 右侧滑入/滑出 |
| 自动滚动 | `scrollIntoView({ behavior: 'smooth' })` | 新消息后自动滚动 |
| 拖拽 | `pointerdown/move/up` 事件 | 保留 proto 实现 |
| SSE 准备 | `EventSource` 或 `fetch` + ReadableStream | P2 用模拟，P3 接 SSE |

### 2.3 实施步骤

**步骤 1：chatStore**

```typescript
export const useChatStore = defineStore('chat', () => {
  const messages = ref<ChatMessage[]>([])
  const isOpen = ref(false)
  const isLoading = ref(false)
  const sessionId = ref(crypto.randomUUID())

  function addMessage(role: 'user' | 'assistant', content: string) {
    messages.value.push({ role, content, timestamp: Date.now() })
  }

  function toggle() { isOpen.value = !isOpen.value }
  function close() { isOpen.value = false }

  return { messages, isOpen, isLoading, sessionId, addMessage, toggle, close }
})
```

**步骤 2：ChatToggle.vue（浮动按钮）**

```vue
<script setup lang="ts">
const store = useChatStore()
const btnRef = ref<HTMLElement | null>(null)
let isDragging = false
let startX = 0, startY = 0

function onPointerDown(e: PointerEvent) {
  isDragging = false
  startX = e.clientX
  startY = e.clientY
  document.addEventListener('pointermove', onPointerMove)
  document.addEventListener('pointerup', onPointerUp)
}

function onPointerMove(e: PointerEvent) {
  if (Math.abs(e.clientX - startX) > 5 || Math.abs(e.clientY - startY) > 5) {
    isDragging = true
  }
}

function onPointerUp() {
  document.removeEventListener('pointermove', onPointerMove)
  document.removeEventListener('pointerup', onPointerUp)
  if (!isDragging) store.toggle()
}
</script>

<template>
  <button
    ref="btnRef"
    class="chat-toggle"
    :class="{ open: store.isOpen }"
    @pointerdown.prevent="onPointerDown"
  >💬</button>
</template>

<style scoped>
.chat-toggle {
  position: fixed;
  bottom: 24px; right: 24px;
  width: 56px; height: 56px;
  border-radius: 50%;
  background: var(--primary);
  color: white;
  font-size: 24px;
  border: none;
  box-shadow: 0 4px 16px rgba(0,0,0,0.2);
  cursor: grab;
  z-index: 1000;
  touch-action: none;
  display: flex; align-items: center; justify-content: center;
}
.chat-toggle.open { display: none; }
</style>
```

**步骤 3：ChatPanel.vue**

```vue
<template>
  <Transition name="slide">
    <div v-if="store.isOpen" class="chat-panel">
      <header class="chat-header">
        <h3>🤖 AI 财报助手</h3>
        <button @click="store.close">✕</button>
      </header>

      <div class="chat-messages" ref="messagesEl">
        <ChatWelcome v-if="store.messages.length === 0"
          :suggestions="['盈利能力怎么样？', '有什么风险？', '现金流健康吗？']"
          @select="sendMessage"
        />
        <ChatMessage v-for="msg in store.messages" :key="msg.timestamp" :message="msg" />
      </div>

      <div class="chat-input">
        <textarea
          v-model="input"
          placeholder="输入你的问题..."
          @keydown.enter.exact.prevent="sendMessage(input)"
          @keydown.enter.shift="input += '\n'"
          rows="1"
          ref="inputEl"
        />
        <button @click="sendMessage(input)" :disabled="!input.trim()">发送</button>
      </div>
    </div>
```
