## 一、PRD（产品需求文档）

### 1.1 背景

KPI 卡片展示了最新一年的快照数据，用户还需要看到近 5 年的趋势变化。速览表格以结构化方式呈现 5 个核心指标 5 年的数据。

### 1.2 需求描述

作为用户，我可以查看 5 个核心指标（营收、净利润、毛利率、资产负债率、现金流）近 5 年的表格数据。

### 1.3 验收标准

| # | 验收条件 |
|---|---------|
| AC1 | 渲染 5 行 × 5 年数据表格（行=指标，列=年份） |
| AC2 | 表头行 sticky 固定（滚动时保持可见） |
| AC3 | 斑马纹交替行背景 |
| AC4 | 移动端支持横向滚动 |
| AC5 | 点击某行 → 切换到该指标的折线图视图 |
| AC6 | 表格和折线图之间可切换（Tab 按钮） |

### 2.3 实施步骤

**步骤 1：后端 timeline 端点**

```java
@GetMapping("/{companyCode}/timeline")
public ApiResponse<TimelineResponse> getTimeline(
    @PathVariable String companyCode,
    @RequestParam(defaultValue = "revenue,profit,grossMargin,debtRatio,cashFlow") String metrics) {

    List<String> metricKeys = Arrays.asList(metrics.split(","));
    // 查询最近 5 年数据
    List<String> years = List.of("2020", "2021", "2022", "2023", "2024");
    List<MetricSeries> series = metricKeys.stream()
        .map(key -> buildMetricSeries(companyCode, key, years))
        .toList();

    return ApiResponse.success(new TimelineResponse(years, series));
}
```

**步骤 2：实现 useTimeline composable**

```typescript
export function useTimeline(companyCode: Ref<string>) {
  const data = ref<TimelineResponse | null>(null)
  const loading = ref(false)
  const currentView = ref<'table' | 'line'>('table')
  const selectedMetric = ref('revenue')

  async function fetch() {
    loading.value = true
    const res = await getReportTimeline(companyCode.value,
      'revenue,profit,grossMargin,debtRatio,cashFlow')
    data.value = res.data.data
    loading.value = false
  }

  watch(companyCode, fetch, { immediate: true })

  return { data, loading, currentView, selectedMetric, fetch }
}
```

**步骤 3：实现 DataTable.vue**

**步骤 4：格式化函数**

```typescript
function formatValue(value: number, unit: string): string {
  if (unit === '%') return `${value.toFixed(1)}%`
  if (unit === '亿') {
    if (value >= 10000) return `${(value / 10000).toFixed(2)} 万亿`
    return `${value.toLocaleString()} 亿`
  }
  return `${value.toLocaleString()} ${unit}`
}
```
