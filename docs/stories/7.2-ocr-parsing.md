## 一、PRD（产品需求文档）

### 1.1 背景

上传的 PDF/图片文件需要经过 OCR 识别提取文本内容，才能进入后续的结构化提取流程。

### 1.2 需求描述

作为系统，我需要从上传的 PDF 和图片文件中提取文本和表格数据，为后续 NER 结构化提取做准备。

### 1.3 验收标准

| # | 验收条件 |
|---|---------|
| AC1 | PDF 文件文本提取准确率 > 95% |
| AC2 | 图片 OCR 识别中文准确率 > 90% |
| AC3 | 表格区域被正确识别并结构化 |
| AC4 | 处理进度可轮询：展示"OCR 识别第 3/12 页" |
| AC5 | 处理失败自动重试最多 3 次 |

### 1.4 处理流程

```
PDF 上传 → MinIO 存储
  → RocketMQ: ocr-parse 队列
  → Python 消费:
      PDF? → pdfplumber (表格) + PyMuPDF (文字)
      图片? → PaddleOCR
  → 文本内容返回 → NER 提取 (Story 7.3)
```

**步骤 2：OCR 服务**

```python
# app/services/ocr_service.py
import fitz  # PyMuPDF
import pdfplumber
from paddleocr import PaddleOCR

class OcrService:
    def __init__(self):
        self.paddle_ocr = PaddleOCR(use_angle_cls=True, lang='ch')

    def process(self, file_url: str, file_type: str, task_id: str):
        """处理上传文件，更新进度到数据库"""
        if file_type == 'pdf':
            return self._process_pdf(file_url, task_id)
        elif file_type in ['jpg', 'jpeg', 'png']:
            return self._process_image(file_url, task_id)
        else:
            raise ValueError(f"Unsupported file type: {file_type}")

    def _process_pdf(self, file_url: str, task_id: str) -> str:
        all_text = []
        # 1. PyMuPDF 提取文字
        doc = fitz.open(file_url)
        total_pages = len(doc)

        for i, page in enumerate(doc):
            text = page.get_text()
            all_text.append(f"--- 第 {i+1} 页 ---\n{text}")

            # 更新进度
            self._update_progress(task_id, "ocr",
                f"OCR 识别第 {i+1}/{total_pages} 页...",
                int((i+1) / total_pages * 80))

        # 2. pdfplumber 提取表格
        with pdfplumber.open(file_url) as pdf:
            for i, page in enumerate(pdf.pages):
                tables = page.extract_tables()
                for j, table in enumerate(tables):
                    table_text = self._table_to_text(table)
                    all_text.append(f"--- 第 {i+1} 页 表格 {j+1} ---\n{table_text}")

        return "\n\n".join(all_text)

    def _process_image(self, file_url: str, task_id: str) -> str:
        self._update_progress(task_id, "ocr", "正在 OCR 识别图片...", 10)
        result = self.paddle_ocr.ocr(file_url, cls=True)
        lines = []
        for line in result[0]:
            text = line[1][0]
            confidence = line[1][1]
            lines.append(f"{text} [置信度: {confidence:.0%}]")
        return "\n".join(lines)

    def _table_to_text(self, table: list[list]) -> str:
        """将 pdfplumber 表格转为 Markdown 表格文本"""
        if not table: return ""
        lines = []
        for row in table:
            cells = [str(cell or "") for cell in row]
            lines.append(" | ".join(cells))
        return "\n".join(lines)

    def _update_progress(self, task_id: str, stage: str, message: str, percent: int):
        """更新任务进度到数据库（通过 HTTP 回调 Java 端）"""
        # 调用 Java 内部 API 更新 mq_task_records 表
        httpx.post(f"{JAVA_URL}/internal/tasks/{task_id}/progress",
                   json={"stage": stage, "message": message, "percent": percent})
```

# app/services/mq_consumer.py
from rocketmq.client import PushConsumer, ConsumeStatus

class OcrTaskConsumer:
    def __init__(self, ocr_service: OcrService):
        self.consumer = PushConsumer('ocr-consumer-group')
        self.consumer.subscribe('ocr-parse', self.callback)
        self.ocr_service = ocr_service

    def callback(self, msg):
        try:
            data = json.loads(msg.body)
            task_id = data['taskId']
            file_url = data['fileUrl']
            file_type = data['fileType']
