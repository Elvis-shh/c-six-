## 一、PRD（产品需求文档）

### 1.1 背景

用户可能一天内分析多家公司，需要快速切换。分析历史提供"最近分析"入口，避免重复搜索。

### 1.2 需求描述

作为用户，我可以查看最近分析过的公司记录，点击即可快速切换，无需重新搜索。

### 1.3 验收标准

| # | 验收条件 |
|---|---------|
| AC1 | Navbar 中显示"最近分析"下拉按钮 |
| AC2 | 点击展开历史下拉列表，显示最近 20 条分析记录 |
| AC3 | 每条记录显示：公司名、股票代码、相对时间（"3 分钟前"） |
| AC4 | 点击历史项直接加载对应公司 Dashboard |
| AC5 | 重复分析同一公司时，仅更新时间戳（去重） |
| AC6 | 刷新页面后历史不丢失（localStorage 持久化） |
| AC7 | 支持单条删除和清空全部历史 |

### 2.1 技术方案

| 技术点 | 选型 | 说明 |

### 2.3 实施步骤

**步骤 1：定义类型**

```typescript
// types/index.ts
export interface AnalysisHistoryItem {
  code: string
  name: string
  timestamp: number  // Date.now()
}
```

**步骤 2：实现 useHistory composable**

```typescript
// composables/useHistory.ts
const STORAGE_KEY = 'smartreport_history'
const MAX_ITEMS = 20

export function useHistory() {
  const items = ref<AnalysisHistoryItem[]>([])

  // 初始化：从 localStorage 加载
  function load() {
    try {
      const raw = localStorage.getItem(STORAGE_KEY)
      items.value = raw ? JSON.parse(raw) : []
    } catch { items.value = [] }
  }

  // 持久化
  function save() {
    localStorage.setItem(STORAGE_KEY, JSON.stringify(items.value))
  }

  // 添加记录（去重 + 限制条数）
  function add(company: { code: string; name: string }) {
    items.value = items.value.filter(i => i.code !== company.code)
    items.value.unshift({ ...company, timestamp: Date.now() })
    if (items.value.length > MAX_ITEMS) items.value = items.value.slice(0, MAX_ITEMS)
    save()
  }

  // 删除单条
  function remove(code: string) {
    items.value = items.value.filter(i => i.code !== code)
    save()
  }

  // 清空全部
  function clear() {
    items.value = []
    save()
  }

  load()  // 立即加载
  return { items, add, remove, clear }
}
```

**步骤 3：实现 useRelativeTime composable**

```typescript
export function useRelativeTime(timestamp: number): string {
  const diff = Date.now() - timestamp
  const seconds = Math.floor(diff / 1000)
  if (seconds < 60) return '刚刚'
  const minutes = Math.floor(seconds / 60)
  if (minutes < 60) return `${minutes} 分钟前`
  const hours = Math.floor(minutes / 60)
  if (hours < 24) return `${hours} 小时前`
  const days = Math.floor(hours / 24)
  if (days < 7) return `${days} 天前`
  return new Date(timestamp).toLocaleDateString()
}
```
