# SmartReport 使用说明

**智能财报研读与 AI 分析助手** — 支持 A 股上市公司财报数据，提供搜索、可视化、AI 问答、文件上传解析和报告导出。

---

## 1. 部署

### 环境要求
- Docker + Docker Compose
- 主机内存 >= 4GB

### api接入
需要手动接入自己的deepseekAPI，或其他模型api。会影响ai的使用效果。
请在/devops/.env中输入自己的API。

### 一键部署

```bash
# 从项目根目录执行
cd devops
docker-compose up -d --build
```

首次构建会拉取镜像并编译前端/后端/AI 引擎，约需 5–10 分钟。

### 服务端口

| 服务 | 地址 | 说明 |
|------|------|------|
| 前端界面 | `http://localhost:3000` | Vue 3 + Nginx |
| 后端 API | `http://localhost:8080` | Spring Boot |
| AI 引擎 | `http://localhost:8000` | Python FastAPI |
| MySQL | `localhost:3307` | 数据库（内部 3306） |
| Redis | `localhost:6379` | 缓存 |

### 停止 / 重启

```bash
docker-compose down          # 停止
docker-compose up -d --build # 重新构建并启动
```

## 2. 使用

### 搜索公司
1. 打开 `http://localhost:3000`，进入搜索首页。
2. 在搜索框中输入公司名称或股票代码（如「贵州茅台」或「600519」），系统自动联想。
3. 也可点击下方 **热门分析** 标签直接进入对应公司。

### 查看分析仪表盘
进入 `/dashboard/{股票代码}` 后可看到：

| 模块 | 说明 |
|------|------|
| **核心指标概览** | 营收、净利润、现金流等 KPI 卡片，含同比变化 |
| **近五年关键数据** | 表格视图，一览多年趋势 |
| **趋势图表** | 折线图，可切换指标对比 |
| **核心指标详解** | 按类别分组（营收、利润、现金流、负债等），含评分 |
| **经营亮点** | 正面指标自动汇总 |
| **风险提示** | 需要关注的财务风险项 |
| **趋势预测** | 基于历史数据的线性回归预测，含 R²评分 |
| **行业对比** | 与行业基准对比 |

## 3. 上传财报文件

支持自行上传财报文件（PDF / Word / Excel / TXT / 图片），由 AI 引擎自动解析并入库。

### 操作步骤
1. 在搜索页或仪表盘顶部找到**上传区域**。
2. 拖拽或点击选择文件（支持格式：`.pdf`(建议) `.doc` `.docx` `.xls` `.xlsx` `.txt` `.jpg` `.jpeg` `.png`，最大 50MB）。
3. 上传后系统自动调用 AI 解析，显示进度条。
4. 解析完成后弹出**提取结果确认面板**：
   - 核对 AI 提取的指标（营收、净利润、毛利率等）
   - 低置信度字段标记为黄色需人工确认
   - 补填公司代码和报告年份
5. 点击「确认并爬取近 5 年年报」，系统后台自动爬取历史数据。
6. 完成后可点击「查看报告」跳转仪表盘。

### 补充说明
- 上传的财报可通过 Navbar 「**财报库**」随时查看和删除。
- 系统内置财报不可删除，仅「用户上传」来源的财报可删除。

## 4. AI 对话

在任何公司仪表盘页面，点击右下角 **AI 助手浮动按钮** 打开聊天面板。

### 使用方式
- 输入自然语言问题（如「这家公司赚钱能力怎么样？」「最大的风险是什么？」）
- AI 会结合财报原文和行业常识，用通俗语言回答，并标注来源。
- 支持 SSE 流式输出（打字机效果），自动滚动到底部。
- 欢迎页提供**建议问题快捷按钮**，点击即可发送。

### 技术说明
- 对话基于 RAG（检索增强生成）+ LLM，先查财报原文段落再生成回答。
- 当前支持的 LLM Provider：DeepSeek / OpenAI / 智谱（可在 `.env` 中配置）。

## 5. 导出报告

在仪表盘页面，Navbar 右侧可见「**导出**」按钮。

### 支持格式

| 格式 | 说明 |
|------|------|
| **PNG** | 单页截图，适合分享 |
| **PDF** | 多页排版，自动附带免责声明 |
| **Word (.doc)** | 可编辑文档 |
| **Excel (.xlsx)** | 结构化数据表格 |

### 导出逻辑
- 前端直接渲染（客户端导出），无需等待后端处理。
- 自动移除上传区、聊天按钮等非报告元素。
- PDF/Word 自动附加「仅供参考，不构成投资建议」免责声明。

## 6. 其他功能

### 6.1 账户系统（注册/登录）
- 支持邮箱注册和登录。
- JWT 令牌认证，accessToken 有效期 2 小时，refreshToken 7 天。
- 未登录可搜索和查看热门公司，**仪表盘需登录**后才能访问。
- 登录后历史记录云端同步，换设备不丢失。

### 6.2 历史记录
- Navbar 「**最近分析**」按钮可查看最近 20 条分析记录。
- 未登录时存储在浏览器本地，登录后自动同步到云端。
- 支持单条删除或全部清空。

### 6.3 财报库
- Navbar 「**财报库**」按钮打开全量财报浏览面板。
- 可按公司代码/名称/年份搜索过滤。
- 区分「系统内置」和「用户上传」来源，后者可删除。

### 6.4 行业对比
- 仪表盘内提供**行业基准对标**功能。
- 将目标公司的核心指标与行业平均水平对比展示。

### 6.5 指标详解与百科
- 每个财务指标附带**名词解释**（「什么是净资产收益率？」等）。
- 帮助非专业用户理解财务术语。

### 6.6 管理与运维接口
- `GET /actuator/health` — 健康检查
- `GET /actuator/metrics` — 指标监控（Prometheus 兼容）
- AI 引擎健康检查：`GET /ai/v1/health`
